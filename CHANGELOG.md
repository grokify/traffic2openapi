## [v0.1.0] - 2026-02-07

### Highlights

- Generate OpenAPI 3.0/3.1/3.2 specifications from HTTP traffic logs captured via HAR files, proxy captures, or test automation
- Multi-source traffic capture with adapters for HAR files, Playwright, Go http.Client, and proxy logs
- Intelligent inference engine with path parameter detection, schema inference, and security/pagination/rate-limit detection
- CLI tools for spec management: diff, merge, serve, and static site generation

### Added

- IR (Intermediate Representation) JSON Schema v1 for HTTP traffic records
- `pkg/ir` package with types, batch/NDJSON I/O, and streaming support
- `pkg/har` package for HAR file parsing and conversion
- `pkg/inference` engine for path template and JSON Schema inference
- `pkg/openapi` generator supporting OpenAPI 3.0.3, 3.1.0, and 3.2.0
- `pkg/sitegen` package for static HTML documentation site generation
- `traffic2openapi` CLI with `convert`, `generate`, and `validate` commands
- `merge` command for combining multiple IR files or OpenAPI specs with deduplication
- `diff` command for comparing OpenAPI specs with breaking change detection
- `serve` command for interactive documentation with Swagger UI or Redoc
- `site` command for generating static HTML documentation from traffic logs
- Watch mode for `generate` command with file change auto-regeneration
- Path parameter detection for UUIDs, numeric IDs, hashes, and dates
- Context-aware path parameter naming with 180+ resource patterns
- Format detection for email, UUID, date-time, URI, IPv4, and IPv6
- Required vs optional field tracking across multiple requests
- Security scheme detection for Bearer (with JWT), Basic auth, and API keys
- Pagination pattern detection for page/limit and cursor-based patterns
- Rate limit header detection (X-RateLimit-*, Retry-After)
- Sensitive header filtering (authorization, cookies, API keys)
- Playwright adapters for Python and TypeScript
- LoggingTransport for capturing HTTP traffic from Go http.Client

### Documentation

- README with quick start, architecture diagram, and API usage examples
- Browser capture guide for Playwright, Cypress, and DevTools
- MkDocs documentation site with comprehensive guides

### Tests

- Unit tests for IR, inference, OpenAPI, and HAR packages
