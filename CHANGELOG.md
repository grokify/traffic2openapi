## [v0.1.0] - 2026-02-07

### Highlights

- Generate OpenAPI 3.0/3.1/3.2 specifications from HTTP traffic logs captured via HAR files, proxy captures, or test automation
- Multi-source traffic capture with adapters for HAR files, Playwright, Go http.Client, and proxy logs
- Intelligent inference engine with path parameter detection, schema inference, and security/pagination/rate-limit detection
- CLI tools for spec management: diff, merge, serve, and static site generation
